Description

  • The Data Template Ontology (TempO) is based on concepts and principles for creating templates from ISO 23387 and the associated XML data schema, which is currently under development.

  • The Data Template Ontology (TempO) ontology maps the data model of the ISO 23387-3 for Data templates for construction objects used in the life cycle of built assets The namespace for TempO terms is https://w3id.org/tempo

    The preferred prefix for the TempO namespace is tempo.
